css需要一个英雄！

下面是以前的css。

body {
  font: 12px Helvetica, Arial, sans-serif;
}
a.button {
  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
  border-radius: 5px;
}

如果我们能省略大括号？

body
  font: 12px Helvetica, Arial, sans-serif;

a.button
  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
  border-radius: 5px;

分号是不是也可以去掉呢？

body
  font: 12px Helvetica, Arial, sans-serif

a.button
  -webkit-border-radius: 5px
  -moz-border-radius: 5px
  border-radius: 5px

何必让冒号继续停留？

body
  font 12px Helvetica, Arial, sans-serif
  
a.button
  -webkit-border-radius 5px
  -moz-border-radius 5px
  border-radius 5px

来干些事看看

border-radius()
  -webkit-border-radius arguments
  -moz-border-radius arguments
  border-radius arguments
  
body
  font 12px Helvetica, Arial, sans-serif
  
a.button
  border-radius(5px)

如何混合书写呢？

border-radius()
  -webkit-border-radius arguments
  -moz-border-radius arguments
  border-radius arguments
  
body
  font 12px Helvetica, Arial, sans-serif
  
a.button
  border-radius 5px

创建与分享

@import 'vendor'

body
  font 12px Helvetica, Arial, sans-serif
  
a.button
  border-radius 5px

甚至我们可以加入语言函数

sum(nums...)
  sum = 0
  sum += n for n in nums
  
sum(1 2 3 4)
// => 10

如果所有可选又如何？

fonts = helvetica, arial, sans-serif

body {
  padding: 50px;
  font: 14px/1.4 fonts;
}

stylus的获取

如果你的电脑已经安装了node.js,那么安装stylus将变得非常容易。若没有安装，可以到这里下载！ 安装完成后，确保它们还包括npm和node的包管理工具。

那么，现在打开node自带的命令行工具，键入如下代码:

$ npm install stylus

等待安装完成即可
